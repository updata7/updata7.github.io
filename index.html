git常用命令

更新远程github库到本地

  • 1.git status – 查看是否有冲突

  • 2.没冲突的话，直接到第三步，有冲突的话，想忽略本地的修改：

    ​ git reset –hard origin/master

    不想忽略本地的修改：

    ​ git add filename

    ​ git commit -m “message”

    如果合并后觉得混乱

    ​ git reset –hard HEAD

  • 3.想合并

    git stash   //备份当前的工作区的内容，从最近的一次提交中读取相关内容，让工作区保证和上次提交的内容一致。同时，将当前的工作区内容保存到Git栈中。
git pull
git stash pop

  • 3.git pull

上传本地的文件

​ git add .

​ git commit -m ‘something’

​ git push

上传大文件 超过100M的

​ 安装 git-lfs

​ brew install git-lfs

​ git lfs track “*.zip” –这里的 “ *.zip “ 就是要上传的大文件的路径

​ git add .gitattributes

​ git add demo.gif

​ git commit -m “xxx”

​ git push

查看某个文件本次改变情况

​ git log -p filename

查看当前分支信息

git config –list

查看处于哪个分支

git branch

创建并切换到新分支

git checkout -b xxx

查看远程仓库地址

git remote -v

撤销git add

git rm –cached filename

如果撤销的是文件夹要加 -r
git rm -r –cached floder

mac-php开发环境

开启Apache服务

mac有自带apache和php 直接打开命令行即可

  • 首先切换到root用户 sudo -i

  • 接着查看80端口是否被暂用 lsof

  • 查看apache 版本 httpd -v

  • 开启apache apachectl start

  • 重启apache apachectl restart

  • 关闭apache apachectl stop

  • 运行 127.0.0.1,显示it works表示apache开启成功

开启php

进入路径/etc/apache2 开启apache配置文件httpd.conf 开启php模块

  • 找到类似这行，然后把注释去掉 LoadModule php7_module libexec/apache2/libphp7.so

在 /etc/apache2/httpd.conf 查看网站服务器根目录

查找 DocumentRoot，能看到网站跟目录

重启Apache服务器，在网站服务器根目录，写个test.php

  • 到浏览器执行 127.0.0.1/test.php，能显示php的相关信息，就是正常的

宝塔

数据库，远程访问不了

终端输入
mysql -uroot -p

输入密码后，以下命令是在mysql内完成，仍然是在终端
mysql> use mysql;
mysql> select user, Host from user;
+————+————+
| user | Host |
+————+————+
| xxxxxxxxxx | % |
| numbers | % |
| root | 127.0.0.1 |
| | 4z-k20-a35 |
| root | 4z-k20-a35 |
| root | ::1 |
| | localhost |
| xxxxxxxxxx | localhost |
| numbers | localhost |
| root | localhost |
+————+————+
注意这里 root 只允许本地连接，最高权限的root用户只允许本地连接，所以其他用户也就只允许本地连接，
所以需要把root修改为允许所有连接，接着执行
mysql> UPDATE user SET Host = ‘%’ WHERE User = ‘root’ AND Host=’localhost’;
mysql> FLUSH PRIVILEGES; – 使修改生效

Linux记录

安装pm2遇到的问题及解决

问题

* 安装命令：npm install pm2 -g

解决

vi /etc/profile # 在PATH后，添加路径 /usr/local/nodejs/bin
source /etc/profile # 让刚才的修改生效

修改远程端口为2202

$ vi /etc/ssh/sshd_config
将Port 22，改为 Port 2202

防火墙允许此端口对外开放
firewall-cmd --zone=public --add-port=2202/tcp --permanent
firewall-cmd --reload   // 重启防火墙，让设置生效

重启ssh服务
systemctl restart sshd.service

git 443 No route to host

vi /etc/hosts/ 加地址 如 xx.xx.xx.xx github.com

gitee 443 No route to host

vi /etc/hosts/ 加地址 如 xx.xx.xx.xx gitee.com

查看端口号运行情况

lsof -i:端口号

添加计划任务

crontab -e
30 1 * * * /home/crontab/mongod_bak.sh #表示每天凌晨1点30执行备份

/sbin/service crond reload #重新载入配置
chkconfig –level 35 crond on #加入开机自动启动:
/sbin/service crond start #启动服务
crontab -l #列出crontab文件
