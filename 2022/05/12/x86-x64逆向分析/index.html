x86/x64逆向分析

汇编语言的两种主流语体： Intel、AT&T

x64 RAX 64位寄存器
有16个通用寄存器 RCX RDX R8 R9 这四个寄存器存放函数参数
8个字节 每个字节8位

  • x86 EAX 32位寄存器
    有8个通用寄存器 4个字节 每个字节8位

  • 16位 AX
    AH 高8位 AL 低8位 2个字节，每个字节8位

操作码 + 操作数 = 指令

lea 装载有效地址
lea rcx, addr

函数序言、函数尾声
x86
序言:
push ebp
mov ebp, esp
sub esp, 0CCh

尾声:
    move esp, ebp
    pop  ebp

逆向生长的堆栈
栈是内存，会存放临时变量会紧要的数据
PUSH 减法操作
POP 加法操作
栈顶：RSP(x64) / ESP(x86)
栈底：EBP

堆 低地址
栈 高地址

EIP 寄存器 存储CPU即将要执行的指令

HEX，Hexadecimal ，十六进制。
DEC，Decimal ，十进制。
OCT，Octal ，八进制。
BIN，Binary ，二进制。

栈的噪音
CALL等价于：PUSH + JMP
RET 等价于：POP + JMP

内存
dd dc du

汇编指令
mov eax, [0x5] ==>> eax = *((int *)0x5)

JE 如果上次比较 A = B 就跳
JMP 无条件跳转
push 5 —–> esp -= 4(32位设备) *((int *) esp) = 5
push eax —–> esp -= 4(32位设备) *(int *) esp) = eax
pop eax —–> esp += 4(32位设备) eax = *((int *) esp)
call 地址 —–> push eip(call的下一条指令); jmp 地址
ret —–> pop eip

Intel CPU 的四个特权级
1、Ring 0 内核层、驱动层
2、Ring 1 未使用
3、Ring 2 未使用
4、Ring 3 用户层
