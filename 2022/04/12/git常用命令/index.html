git常用命令

更新远程github库到本地

  • 1.git status – 查看是否有冲突

  • 2.没冲突的话，直接到第三步，有冲突的话，想忽略本地的修改：

    ​ git reset –hard origin/master

    不想忽略本地的修改：

    ​ git add filename

    ​ git commit -m “message”

    如果合并后觉得混乱

    ​ git reset –hard HEAD

  • 3.想合并

    git stash   //备份当前的工作区的内容，从最近的一次提交中读取相关内容，让工作区保证和上次提交的内容一致。同时，将当前的工作区内容保存到Git栈中。
git pull
git stash pop

  • 3.git pull

上传本地的文件

​ git add .

​ git commit -m ‘something’

​ git push

上传大文件 超过100M的

​ 安装 git-lfs

​ brew install git-lfs

​ git lfs track “*.zip” –这里的 “ *.zip “ 就是要上传的大文件的路径

​ git add .gitattributes

​ git add demo.gif

​ git commit -m “xxx”

​ git push

查看某个文件本次改变情况

​ git log -p filename

查看当前分支信息

git config –list

查看处于哪个分支

git branch

创建并切换到新分支

git checkout -b xxx

查看远程仓库地址

git remote -v

撤销git add

git rm –cached filename

如果撤销的是文件夹要加 -r
git rm -r –cached floder
