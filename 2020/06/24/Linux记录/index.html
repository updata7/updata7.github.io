Linux记录

安装pm2遇到的问题及解决

问题

* 安装命令：npm install pm2 -g

解决

vi /etc/profile # 在PATH后，添加路径 /usr/local/nodejs/bin
source /etc/profile # 让刚才的修改生效

修改远程端口为2202

$ vi /etc/ssh/sshd_config
将Port 22，改为 Port 2202

防火墙允许此端口对外开放
firewall-cmd --zone=public --add-port=2202/tcp --permanent
firewall-cmd --reload   // 重启防火墙，让设置生效

重启ssh服务
systemctl restart sshd.service

git 443 No route to host

vi /etc/hosts/ 加地址 如 xx.xx.xx.xx github.com

gitee 443 No route to host

vi /etc/hosts/ 加地址 如 xx.xx.xx.xx gitee.com

查看端口号运行情况

lsof -i:端口号

添加计划任务

crontab -e
30 1 * * * /home/crontab/mongod_bak.sh #表示每天凌晨1点30执行备份

/sbin/service crond reload #重新载入配置
chkconfig –level 35 crond on #加入开机自动启动:
/sbin/service crond start #启动服务
crontab -l #列出crontab文件
